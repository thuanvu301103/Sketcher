# Sketcher
Plug-in for Blender

## Activate Virtual Environment
- In project folder, create a Virtual Environment:
```bash
python -m venv venv
```
- Activate Virtual Environment:
```bash
venv\Scripts\activate
```

## Run Project
```bash
python main.py
```

## Create Object in Blender
- Open Blender ```Scripting``` mode
- Open the ```mesh.py``` script
- Run the script
